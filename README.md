# AI Interior Room Designer 🛋️

An AI-powered web application that helps users redesign and decorate their rooms. Built with Streamlit and OpenAI Vision/Image APIs.

## Features
- **Image Understanding:** Analyze room type, furniture, and lighting using AI Vision.
- **Style Presets:** Choose from Minimal, Modern Luxury, Scandinavian, and more.
- **AI Design Generation:** Generate high-quality redesigned room images based on your style preference.
- **Design Recommendations:** Get furniture suggestions, color palettes, and layout advice.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Set your `OPENAI_API_KEY` in environment variables or a `.env` file.
3. Run the app: `streamlit run app.py`

## Deployment
This app is ready to be deployed on Streamlit Community Cloud. Make sure to set your `OPENAI_API_KEY` in the app's Secrets settings.
